# Train Delay & Passenger Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes train delays, passenger traffic,
revenue, and route performance.

## Tools Used
- Power BI
- Excel
- DAX
- Power Query

## KPIs
- Total Passengers
- Average Delay Time
- Revenue
- On-Time Performance %

## Insights
1. Route A had the highest passenger traffic.
2. Delays increased during monsoon months.
3. Weekend revenue was higher than weekdays.
